La page web est obtenue par interprétation d'un code HTML envoyé par le serveur en réponse à la requête qui lui est envoyée (adresse en http://www.ect). C'est le navigateur qui interprète le code.

Il existe 2 sortes de balises:

  1. Les balises les plus courantes encandrent du code qui se termine par une balise dite fermante. Exemple: </balise> Contenu HTML
    2. </balise>
  2. les balises dite orphelines qui n'encardent aucun code et qui se ferment toutes seules. Exemple: <br/> ; <hr/> ; <img/> ; <input/> ; ect...

  • u
  • em
